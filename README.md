### CRUD
users
- **GET**/me -> 내 정보 조회
- **POST**/register -> 회원가입

auth
- **POST**/login -> 로그인

-------------------------
#### 회원가입
- 아이디(이메일)
  - 중복확인
  - 필수입력
  - 이메일 형식 입력
- 패스워드
  - 필수입력
  - 비밀번호 확인(일치/불일치)
- 닉네임
  - 중복확인
  - 필수입력
- 이미지
  - 기본 이미지 -> default 이미지 변경 해야 됨(지금은 임의로 넣어둠)
  - 이미지 변경

- 이미지
