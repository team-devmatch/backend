# backend

Spring Boot 기반 REST API 서버

## 기술 스택
- Java 17
- Spring Boot
- Spring Security + JWT
- JPA & Spring Data JPA
- PostgreSQL / Redis
- Kafka (서비스 간 통신)
- Spring Cloud Gateway (API Gateway)
- Swagger (API 문서화)

## 주요 기능
- 회원가입 / 로그인 (JWT 인증)
- 축제 등록 / 조회 / 수정 / 삭제
- Rate Limiting (Redis 연동)
- AI 서비스 연동

## 브랜치 규칙
- feature/기능명 → 기능 개발
- fix/버그명 → 버그 수정
- main → 최종 배포 브랜치
